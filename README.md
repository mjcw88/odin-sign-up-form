# The Odin Project (Full Stack Javascript) - Sign-up Form

Sign-up form project for the Odin Project Full Stack Javascript course.

---

##### HTML features used:
- Form creation
- Form labels
- Form input types (text/phone number/email/password)
- Form validation
- Submit button

---

##### CSS features used:

- External fonts via ```@import``` and ```@font-face```
- ```:root``` to initialise variables
- Dynamic/responsive unit types such as ```rem```
- Mobile responsiveness via ```@media```
